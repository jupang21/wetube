자바 스크립트는 브라우저에 내장되어있다. 
(자바 스크립트는 브라우저에 종속되어있음, 브라우저에 맞춰 설계된 언어)

프론트 개발에는 무조건 javascript 사용 (이것밖에 없어서)
백엔드에는 선택할 수 있음

<<Node.js 란>>
Node.js 는 JS를 브라우저 밖으로 가지고 나와서 사용자의 컴퓨터에서 돌아가게 하는것.
== 브라우저 밖의 JavaScript
JavaScript를 이용해서 브라우저와는 별도로 작동하는 것을 만드는 것!

서버 :: 인터넷에 연결된 컴퓨터 (물리적으로 늘 켜져있는 컴퓨터라는 의미)
소프트웨어적인 서버의 뜻 :: 인터넷에 연결된(네트워크에 연결된) 한 덩어리의 코드
			URL에 응답하고 접속을 허락하는 일
			인터넷에 연결된 내 접속요청에 응답하는 컴퓨터

Express.js
Express.js 는 프레임워크다
(이미 많은 줄의 코드로 만들어져서,누군가만들어 놓아서
어떻게 사용하는지 배우고 나면 우리가 원하는걸 빠르게 만들 수 있는것)
ex)
Django는 Python으로 된 프레임워크
Rails는 Ruby로 된 프레임워크
Laravel은 PHP로 된 프레임워크

Json은 Javascript에서 정보를 담는 방식
